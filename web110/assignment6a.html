Summary:

In this exercise, I learned about the iframe and img HTML tags. The iframe tag allows you to embed videos from external sources such as YouTube (though it is important that you use the version of the YouTube link containing /embed so that it actually functions). By contrast, the img tag allows you to embed an image—from either an internal or external source—into a webpage. Where I see these tags being the most useful is for integrating media into an online portfolio. That would look something along the lines of embedding an image of myself or projects I’ve worked on, as well as any related videos which are already hosted elsewhere.
