I learned how to style a website using CSS. For the purposes of keeping this relatively short, I will focus on the implementation I feel is most useful—the external style sheet. Within a CSS file, each element you wish to manipulate the properties of would be specified as follows:
element {
property_name: property;
}
Though this file defines a style which can be used on any HTML page, a link to it must be included in the head section of the page for it to have any effect. This would look like: <link rel="stylesheet" href="stylesheet_file.css">. Ultimately, I foresee the ability to more granularly manipulate the presentation of my website as a means of making my previously mentioned hobby website and/or professional portfolio more aesthetically pleasing, better organized, and more readable.
