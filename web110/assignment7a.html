My idea for a final project is to focus on the topic of HTML. I would like to use it as an opportunity to establish what HTML is, explain some of the ways it can be used, touch on the formatting of an HTML document (primarily what an HTML document requires as a bare minimum and how tags must be written), and go over some of the HTML elements one can use. Currently, I think the elements I should include are headings, paragraphs, images, audio, and videos (local files as well as ones hosted on other services). Ideally, I would be taking the time to demonstrate each of these element types. As a final inclusion, I would be inclined to briefly talk about how CSS can take the usability (accessibility especially!) and appearance of HTML documents to the next level. That is, if time permits me to do so.

Learning Summary for Assignment 7a:

While completing this activity, I learned how to create a Gantt chart. This is a graphic used in project management for establishing and conveying a timeline for both a project and its individual tasks. This is certainly a tool that lends itself primarily to a professional context, so I believe it will come in handy for my future IT career when I must convey to my superiors and team members the timeline of a project. Additionally, working with a Gantt chart has given me enough exposure to confidently understand one if one were to be presented to me in the future.
