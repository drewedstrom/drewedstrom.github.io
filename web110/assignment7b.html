Learning Summary

In this lesson, what I learned had less to do with actual databases (as I’ve worked with them plenty before) and more to do with the <pre> tag in HTML. See, I usually start these summaries by typing them up into Microsoft Word and then copy them over into the relevant HTML document. This usually doesn’t present too much of an issue as all of my summaries have been simple paragraphs thus far. However, this time I had formatted some of the answers to the activity as these sorts of tables that are made up of new lines and tabs. I certainly wasn’t too keen on playing around with tab spacing in CSS or having to recreate my already formatted text with the use of HTML tags. Then I learned about the <pre> tag, which allows me to format the text in VSCode and maintain that formatting when viewing the webpage. This will certainly save me time in the future if I desire to add text I’ve already formatted to an HTML document without having to put in additional effort of reformatting it.
