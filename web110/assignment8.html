  • UPDATE Customers SET ContactName = 'Alfred Schmidt', City= 'Frankfurt' WHERE CustomerID = 1;

Assignment Summary

In this lesson, I learned a few SQL commands for extracting and modifying records from a table. As to not get too into the weeds, I will focus on commands that concern fetching records. To query records, SELECT is used. The names of columns (or * to select all columns in a table) are placed after SELECT, which is then followed by FROM and the name of the table you want to pull data from. If I desired to pull records that fulfill a certain condition—such as customers whose country is the United States—I could tack on the WHERE clause. For the stated example, that would look like SELECT Customers WHERE Country=’USA’. With these commands at my disposal, I will be able to locate relevant records in an SQL database when necessary as an IT technician.
