Assignment 5a

I learned how to create an index for my website using a combination of an unordered list and the <a> tag with the href attribute. To create an unordered list, the <ul> tag is used. For individual list items within this list, they are denoted by the <li> tag. By themselves, these two tags are used to create a list of pages on your website. Not terribly useful on its own. To make it so the items in that list can be interacted with as to take you to the pages they mention, the <a> tag with the href attribute is used. In the case of this exercise, the href attribute is provided a relative URL for navigating to another page on my website. If I wanted to redirect to a different site, I would use an absolute URL here instead. Where I envision these skills being the most useful is in creating a navigable website as a hobby project or for creating a portfolio to showcase my skills and projects to prospective employers.
