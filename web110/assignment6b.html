Assignment 6b

In this exercise, I learned how to effectively describe images using alt text and ensure acceptable contrast between elements on a website. For good alt text, the rule of thumb is to keep it concise while still being descriptive. For my purposes, this meant changing the alt text of my 8-ball image from “An 8-ball” to “Black billiard ball with the number 8.” By making this change, I am communicating details about the 8-ball that were previously left to the assumption the user is already familiar with the appearance of an 8-ball. In addition to the use of alt text, ensuring proper contrast between different elements of the site and the background is also extremely important for accessibility. For achieving this, the WebAIM Contrast Checker can be utilized. With these new skills, I can be sure that any websites or portfolios I create are always as accessible as they can possibly be.
